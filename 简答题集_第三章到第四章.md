# 简答题集（续）

## 第三章 脂类

**7. 简述油脂自动氧化的机理及其影响因素。**

答：自动氧化是自由基链反应，分为三个阶段：

（1）诱导期：
- 自由基引发：RH → R• + H•
- 需要能量或催化剂引发

（2）传播期：
- R• + O₂ → ROO•
- ROO• + RH → ROOH + R•
- 链式反应不断进行

（3）终止期：
- R• + R• → R-R
- R• + ROO• → ROOR
- ROO• + ROO• → 非自由基产物

影响因素：
- 脂肪酸组成：不饱和脂肪酸易氧化
- 氧浓度：氧浓度高加速氧化
- 温度：温度升高加速氧化
- 光和射线：促进自由基生成
- 金属离子：催化氧化（Fe²⁺、Cu²⁺等）
- 抗氧化剂：延缓氧化

**8. 比较油脂的氢化、酯交换和分提三种改性方法。**

答：

| 方法 | 原理 | 目的 | 应用 |
|------|------|------|------|
| 氢化 | 不饱和脂肪酸双键加氢 | 提高熔点，改善稳定性 | 人造奶油、起酥油 |
| 酯交换 | 脂肪酸在甘油位点重新分布 | 改变熔融特性，改善功能性 | 糖果脂、专用油脂 |
| 分提 | 利用不同组分熔点差异分离 | 获得不同熔点的组分 | 可可脂代用品、月桂酸油 |

**9. 什么是油脂的塑性？影响塑性的因素有哪些？**

答：塑性是指油脂在外力作用下变形，撤去外力后保持变形的性质。

影响因素：
（1）固态脂肪指数（SFI）：
- SFI过高：坚硬，口感差
- SFI过低：过软，易熔化
- 理想范围：10-20

（2）脂肪的晶体形态：
- α型：最不稳定，熔点最低
- β'型：中等稳定，塑性好
- β型：最稳定，塑性差

（3）温度：温度影响晶型转变

## 第四章 蛋白质

**10. 简述蛋白质的结构层次及其特点。**

答：

（1）一级结构：
- 氨基酸的排列顺序
- 肽键连接
- 决定蛋白质的高级结构

（2）二级结构：
- α-螺旋：右手螺旋，3.6个氨基酸/圈
- β-折叠：平行或反平行排列
- β-转角：改变肽链方向
- 无规卷曲：不规则结构

（3）三级结构：
- 整条肽链的空间折叠
- 疏水相互作用、氢键、离子键等稳定
- 形成特定的三维结构

（4）四级结构：
- 多条肽链（亚基）的组装
- 亚基间非共价结合
- 具有生物学功能

**11. 蛋白质有哪些重要的功能性质？**

答：

（1）水合性质：
- 吸水性、保水性
- 影响食品的质地和口感

（2）溶解性：
- 受pH、离子强度、温度影响
- 在等电点时溶解度最低

（3）乳化性：
- 降低界面张力
- 形成稳定的乳化体系

（4）起泡性：
- 降低表面张力
- 形成稳定的泡沫结构

（5）凝胶性：
- 蛋白质分子交联形成网络
- 保持水分和其他成分

（6）黏度：
- 影响食品的口感和质感

（7）组织化：
- 形成纤维状结构
- 模拟肉制品质地

**12. 什么是蛋白质变性？引起变性的因素有哪些？**

答：蛋白质变性是指蛋白质的天然构象被破坏，生物学活性丧失，但一级结构不变。

引起变性的因素：

（1）物理因素：
- 加热：破坏氢键和疏水相互作用
- 冷冻：冰晶形成，pH变化
- 机械处理：剪切力破坏结构
- 高压：影响非共价键
- 辐射：破坏化学键

（2）化学因素：
- 酸碱：影响离子键和氢键
- 有机溶剂：破坏疏水相互作用
- 表面活性剂：破坏疏水核心
- 重金属盐：与巯基等结合
- 尿素、胍盐：破坏氢键

变性应用：
- 烹饪：改善口感和消化性
- 灭菌：使酶和毒素失活
- 加工：改善功能性质
