# 食品化学题库

基于《食品化学（第三版）》（王璋主编）及课程PPT生成的考试题库

## 题库简介

本题库针对食品化学课程考试精心编写，涵盖考试所需的各种题型，按照章节和难度进行分类，适合复习和自测使用。

### 考试题型及分值比例
- **选择题**：30%
- **填空题**：20%
- **结构命名题**：10%
- **简答题**：40%

## 题库特色

- ✅ **题型全面**：覆盖所有考试题型
- ✅ **难度分级**：基础题和提高题分开
- ✅ **章节完整**：按教材章节组织
- ✅ **答案详细**：提供详细解析
- ✅ **格式规范**：Markdown格式，易于编辑

## 快速开始

1. **浏览题目**：查看 [索引.md](索引.md) 快速找到所需题目
2. **按章节学习**：进入相应题型目录，按章节复习
3. **自我测试**：随机抽取题目组成试卷
4. **对照答案**：检查学习效果

## 使用建议

### 复习策略

1. **第一阶段**：按章节学习，理解基本概念
2. **第二阶段**：做基础题，巩固知识点
3. **第三阶段**：做提高题，提升综合能力
4. **第四阶段**：模拟考试，查漏补缺

### 重点章节

- ⭐⭐⭐ 第一章：水（水分活度是核心）
- ⭐⭐⭐ 第二章：碳水化合物（淀粉的糊化与老化）
- ⭐⭐⭐ 第三章：脂类（自动氧化机理）
- ⭐⭐⭐ 第四章：蛋白质（变性与功能性质）
- ⭐⭐ 第五章：维生素与矿物质
- ⭐⭐ 第六章：酶
