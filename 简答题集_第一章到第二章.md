# 简答题集

## 第一章 水

**1. 简述食品中水的存在状态及其特点。**

答：食品中的水主要以两种状态存在：

（1）结合水（束缚水）：
- 与食品中的非水成分通过氢键等作用紧密结合
- 不易冻结（冰点可降至-40℃）
- 不能作为溶剂
- 不易蒸发
- 不能被微生物利用

（2）自由水（游离水）：
- 与普通水性质相似
- 可作为溶剂
- 容易蒸发
- 0℃可以结冰
- 能被微生物利用

结合水又可分为：化合水、邻近水、多层水三种类型。

**2. 什么是水分活度？它对食品稳定性有何影响？**

答：水分活度（Aw）是指食品的水蒸气分压与同温度下纯水的饱和蒸气压之比。

对食品稳定性的影响：
（1）微生物生长：Aw<0.6时大多数微生物不能生长
（2）酶促反应：Aw降低，酶活性下降
（3）非酶褐变：中等水分活度（Aw 0.6-0.7）时反应最快
（4）脂质氧化：Aw过低或过高都会加速氧化
（5）维生素保存：低水分活度有利于维生素保存

**3. 解释水分吸附等温线（MSI）的三个阶段。**

答：水分吸附等温线（MSI）描述食品水分含量与水分活度的关系，分为三个阶段：

（1）第一阶段（Aw<0.2）：
- 单分子层吸附
- 水分子与食品成分直接结合
- 相当于BET理论中的单层吸附

（2）第二阶段（0.2Aw<0.85）：
- 多分子层吸附
- 水分子形成多层结构
- 相当于BET理论中的多层吸附

（3）第三阶段（Aw>0.85）：
- 毛细管凝结
- 自由水开始大量出现
- 食品容易发生腐败变质

## 第二章 碳水化合物

**4. 比较直链淀粉和支链淀粉的结构与性质差异。**

答：

| 特征 | 直链淀粉 | 支链淀粉 |
|------|----------|----------|
| 分子结构 | 线性分子，少量分支 | 高度分支 |
| 糖苷键 | α-1,4糖苷键 | α-1,4和α-1,6糖苷键 |
| 分子量 | 较小（1-20万） | 很大（数百万） |
| 溶解性 | 溶于热水 | 不溶于热水 |
| 糊化性质 | 易糊化 | 糊化温度较高 |
| 老化性质 | 易老化 | 不易老化 |
| 碘显色 | 蓝色 | 红紫色 |

**5. 什么是淀粉的糊化和老化？影响它们的因素有哪些？**

答：

（1）糊化：
淀粉颗粒在热水中吸水膨胀，晶体结构破坏，黏度增加的现象。
影响因素：
- 温度：达到糊化温度（60-80℃）
- 水分：需要足够水分（>30%）
- pH：中性条件最适宜
- 盐类：某些盐可促进糊化

（2）老化：
糊化后的淀粉在低温下重新排列形成结晶的过程。
影响因素：
- 温度：2-4℃最易老化
- 水分：30-60%最易老化
- 淀粉类型：直链淀粉易老化
- pH：中性易老化
- 表面活性剂：可抑制老化

**6. 简述美拉德反应的历程及其对食品的影响。**

答：美拉德反应是还原糖与氨基酸之间的非酶褐变反应，分为三个阶段：

（1）初始阶段：
- 还原糖与氨基酸缩合形成席夫碱
- 重排形成Amadori产物

（2）中期阶段：
- Amadori产物降解
- 产生风味物质

（3）最终阶段：
- 聚合形成类黑精
- 产生褐变色泽

对食品的影响：
正面：产生风味和色泽，如面包烘焙、肉类加工
负面：产生有害物质，降低营养价值
