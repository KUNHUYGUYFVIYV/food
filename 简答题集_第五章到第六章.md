# 简答题集（续）

## 第五章 维生素与矿物质

**13. 比较水溶性维生素和脂溶性维生素的特点。**

答：

| 特征 | 水溶性维生素 | 脂溶性维生素 |
|------|--------------|--------------|
| 溶解性 | 溶于水 | 溶于脂肪和有机溶剂 |
| 吸收 | 易吸收 | 需要胆汁酸帮助吸收 |
| 排泄 | 易随尿液排出 | 不易排泄，易蓄积 |
| 毒性 | 毒性较小 | 过量摄入易中毒 |
| 稳定性 | 多数对热、氧、光敏感 | 对热稳定，但易被氧化 |
| 储存 | 体内储存少 | 可在肝脏和脂肪组织储存 |
| 代表 | 维生素B族、C | 维生素A、D、E、K |

**14. 维生素C有哪些重要的生理功能？在食品加工中如何保持其稳定性？**

答：

生理功能：
（1）抗氧化作用：清除自由基
（2）促进胶原蛋白合成
（3）促进铁的吸收
（4）提高免疫力
（5）参与神经递质合成

保持稳定性的方法：
（1）避光保存
（2）降低温度
（3）调节pH（酸性条件较稳定）
（4）添加抗氧化剂
（5）采用真空或充氮包装
（6）避免与金属离子接触

**15. 食品中矿物质生物利用率的影响因素有哪些？**

答：

（1）矿物质存在形态：
- 有机态矿物质通常比无机态更易吸收
- 血红素铁比非血红素铁吸收率高

（2）食品成分的影响：
- 维生素C促进铁的吸收
- 植酸、草酸抑制矿物质吸收
- 膳食纤维降低矿物质生物利用率

（3）人体因素：
- 年龄：婴幼儿和老年人吸收率高
- 性别：女性铁的需要量更高
- 健康状况：缺铁时吸收率增加

（4）矿物质间的相互作用：
- 钙、铁、锌之间存在竞争抑制

## 第六章 酶

**16. 简述米氏动力学的基本原理及意义。**

答：

米氏方程：v = Vmax[S]/(Km + [S])

基本原理：
（1）酶促反应形成酶-底物复合物
（2）反应分两步进行：E + S ⇌ ES → E + P
（3）Km值表示酶与底物的亲和力

意义：
（1）Km值可反映酶与底物的亲和力（Km越小，亲和力越大）
（2）Vmax表示酶完全被底物饱和时的最大反应速度
（3）可用来比较不同酶的催化效率
（4）为酶抑制剂研究提供理论基础

**17. 食品工业中常用的酶有哪些？举例说明其应用。**

答：

（1）淀粉酶：
- 应用：酿酒、制糖、面包制作
- 作用：水解淀粉生成葡萄糖、麦芽糖

（2）蛋白酶：
- 应用：肉类嫩化、奶酪制作、啤酒澄清
- 作用：水解蛋白质生成肽和氨基酸

（3）脂肪酶：
- 应用：油脂改性、风味改善
- 作用：水解脂肪生成脂肪酸和甘油

（4）果胶酶：
- 应用：果汁澄清、葡萄酒生产
- 作用：水解果胶，降低黏度

（5）葡萄糖异构酶：
- 应用：高果糖浆生产
- 作用：将葡萄糖转化为果糖

**18. 什么是酶的固定化？有哪些方法？**

答：固定化酶是指将酶固定在载体上或限制在一定空间内，使其能反复使用的技术。

方法：
（1）吸附法：
- 通过物理吸附将酶固定在载体表面
- 优点：操作简单
- 缺点：酶易脱落

（2）包埋法：
- 将酶包埋在凝胶网格或微胶囊中
- 优点：条件温和
- 缺点：底物扩散受限

（3）共价结合法：
- 酶与载体通过共价键结合
- 优点：结合牢固
- 缺点：可能破坏酶活性中心

（4）交联法：
- 用双功能试剂使酶分子间交联
- 优点：稳定性高
- 缺点：酶活性损失较大

## 综合应用题

**19. 请设计一个实验测定食品中的水分活度，并说明原理。**

答：

实验设计：

方法一：水分活度仪法
（1）原理：通过测量食品上方空气的相对湿度来确定水分活度
（2）步骤：
   - 将样品放入密闭样品皿
   - 平衡一定时间（通常30分钟）
   - 仪器自动测定样品上方空气的相对湿度
   - 直接读取Aw值

方法二：扩散法
（1）原理：利用不同盐饱和溶液的恒定相对湿度
（2）步骤：
   - 准备一系列不同Aw的盐饱和溶液
   - 将食品样品与盐溶液同置于干燥器中
   - 恒温放置，定时称重
   - 样品质量不变时对应的盐溶液Aw即为样品Aw

**20. 论述水分活度、pH和温度对食品稳定性的综合影响。**

答：

（1）水分活度的影响：
- Aw<0.6：微生物不能生长，酶活性受抑制
- Aw 0.6-0.7：部分耐旱微生物可生长
- Aw>0.9：大多数微生物可生长，食品易腐败

（2）pH的影响：
- 酸性食品（pH<4.5）：抑制大多数细菌生长
- 中性食品（pH 6-7）：微生物最易生长
- 碱性食品（pH>8）：部分细菌可生长

（3）温度的影响：
- 低温（<10℃）：抑制微生物生长
- 中温（25-40℃）：微生物最适生长温度
- 高温（>60℃）：杀死微生物

综合影响：
- 低水分活度+酸性+低温：食品最稳定
- 高水分活度+中性+适温：食品最易腐败
- 可通过控制这三个因素来延长食品保质期
